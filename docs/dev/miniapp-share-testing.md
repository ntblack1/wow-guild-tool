# 小程序手机内测说明

本说明用于把 `wow-guild-tool` 做到可以通过微信小程序在手机上体验测试。

## 当前结论

如果只是微信开发者工具模拟器或同机预览，建议优先使用电脑局域网 IP，例如当前开发机的 `http://192.168.50.101:3000`。

如果要分享给朋友用手机测试，手机必须能访问后端。正式体验版或正式版建议使用公网 HTTPS 后端地址，例如：

```text
https://api.example.com
```

手机里的 `127.0.0.1` 是手机自己，不是开发电脑。

## 最小内测路径

1. 准备可访问的后端地址。
   - 同一 Wi-Fi 临时测试可以用电脑局域网 IP，例如 `http://192.168.1.23:3000`。
   - 体验版/正式版建议使用 HTTPS 公网地址。
2. 微信开发者工具打开项目。
3. 上传为体验版。
4. 在微信公众平台添加体验成员。
5. 体验成员扫码或从体验版入口打开。
6. 打开小程序“调试”页。
7. 在“后端地址”里填写后端 API 地址。
8. 点击“保存地址”。
9. 点击“测试后端健康检查”。
10. 健康检查成功后，去“我的”页选择成员、新增角色，再去活动详情报名。

## 正式发布需要补齐

正式发布或更稳定的体验测试，建议补齐：

1. 后端部署到公网服务器。
2. 使用 HTTPS。
3. 在微信公众平台配置 request 合法域名。
4. 后端数据库迁移和 seed/初始化策略。
5. 不再依赖本地电脑启动后端。
6. 后续接入微信登录，替换当前的内测成员选择方案。

## 当前不是正式登录方案

当前小程序仍使用“我的页选择当前成员”的内测方案。它适合小范围工会成员测试流程，不适合公开发布给大量用户。
