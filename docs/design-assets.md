# 视觉素材需求

## 已生成素材

- `apps/web/src/assets/guild-hero.png`
  - 用途：首页 Hero 主视觉
  - 风格：原创轻奇幻工会接待官，明亮、温暖、非官方素材
  - 生成方式：内置 imagegen

## 已接入的用户提供素材

- `apps/web/src/assets/member-fire-mage.png`
- `apps/web/src/assets/member-paladin.png`
- `apps/web/src/assets/member-priest.png`

用途：首页“工会成员图鉴”。页面通过卡片裁切展示角色主体，并加入“轻微腹肌训练痕迹”的成员设定。

注意：这些素材来自用户本地提供的图片，其中可能包含第三方标识、版权字样或类似官方视觉元素。正式公开部署前，建议替换为无 Logo、无版权字样、完全原创或已授权的版本。

## 后续建议补充

- 角色头像组：4 到 6 个，职业/职责差异明显
- 活动空状态插画：适合“暂无活动”
- 论坛空状态插画：适合“还没人发帖”
- 战报空状态插画：适合“战报待更新”
- 吉祥物图标：可用于公告、黑红榜、按钮点缀
- App 图标 / Logo：适合 Cloudflare Pages favicon 和分享图

## 素材约束

- 必须原创
- 不使用魔兽世界官方素材、Logo、截图、角色或可识别设定
- 不要厚重暗黑，不要过度二次元幼态，不要性感化
- 保持清新明亮、轻奇幻、移动端友好
