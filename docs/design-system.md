# 八块腹肌工会大厅设计系统

## 定位

风格关键词：Apple-like minimal UI、light fantasy、cute guild hall、mobile-first。

第一版避免厚重暗黑风和传统后台感，使用明亮、温暖、干净的轻奇幻社区产品气质。

## 组件规则

- 卡片圆角：`22px` 左右
- 按钮圆角：`16px` 到 `20px`
- 移动端使用底部 Tab，桌面端使用顶部导航
- hover 使用轻微上浮，active 使用轻微按压
- 组件尽量柔和，不使用硬表格线
- 页面留白要足，文字层级清楚
