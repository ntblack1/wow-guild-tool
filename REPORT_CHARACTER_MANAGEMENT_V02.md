# REPORT_CHARACTER_MANAGEMENT_V02

执行日期：2026-07-03

## 1. 目标

在小程序“我的”页提供当前成员的角色管理能力，让成员报名活动前可以先维护自己的游戏角色。

本功能仍属于内测阶段能力，不包含微信登录、手机号授权或复杂权限。

## 2. 已实现功能

- 我的页显示当前成员角色列表。
- 每个角色显示：
  - 角色名
  - 职业
  - 专精
  - 职责
  - 装等
  - 是否主号
- 我的页支持新增角色。
- 我的页支持编辑角色。
- 我的页支持设置主号。
- 我的页支持删除角色，删除前二次确认。
- 保存、删除、设置主号后会刷新角色列表。
- 未选择当前成员时提示先选择成员。
- 没有角色时显示“暂无角色，请先新增角色”。
- 报名页继续依赖当前成员的角色列表。

## 3. 后端接口

新增或完善：

- `POST /members/:id/characters`
- `PATCH /members/:id/characters/:characterId`
- `POST /members/:id/characters/:characterId/set-main`
- `DELETE /members/:id/characters/:characterId`

字段：

- `name`
- `className`，接口层映射到数据库字段 `class`
- `spec`
- `roleType`：`tank` / `healer` / `melee` / `ranged`
- `itemLevel`
- `isMain`

规则：

- `name`、`className` 不能为空。
- `roleType` 必须是允许值。
- `itemLevel` 如果填写，必须是正整数。
- 同一个成员最多只有一个主号；设置新主号时会自动取消其他角色的主号状态。

## 4. 小程序使用方式

成员第一次使用：

1. 打开“我的”页。
2. 在“切换当前成员”中选择自己。
3. 在“我的角色”中点击“新增角色”。
4. 填写角色名、职业、专精、职责、装等。
5. 如这是常用角色，打开“设为主号”。
6. 保存角色。
7. 回到活动详情页点击报名。
8. 报名页会读取当前成员的角色列表，选择角色后提交报名。
